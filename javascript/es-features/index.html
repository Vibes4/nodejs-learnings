Modern JS (ES6+) JavaScript

The features added since ES2015 that you're expected to use idiomatically. They make code shorter, safer against null/undefined, and more expressive.

Destructuring (with defaults)

Unpack arrays and objects, supply defaults for missing values, rename, and read nested structures: const { a = 1, b: renamed, nested: { c } = {} } = obj. Defaults apply only when the value is undefined (not null).

Optional chaining ?.

obj?.a?.b short-circuits to undefined the moment a link is null or undefined, instead of throwing. Works for calls (fn?.()) and indexes (arr?.[0]).

Gotcha: ?. only guards null/undefined, not other falsy values. And don't over-use it — silencing every access can hide real bugs.

Nullish coalescing ??

a ?? b returns b only when a is null or undefined. Unlike ||, it does not treat 0, '', or false as missing — exactly what you want for defaults on numbers and booleans.

Template literals

Backtick strings with ${expr} interpolation and real multi-line support. Tagged templates (tag`...`) let a function process the parts — used for i18n, SQL builders, styled-components.

Default & rest parameters

function f(a, b = 10, ...rest) — b defaults when undefined; rest gathers extra args into a real array (replacing the old arguments object).

Map & Set

  • Map — keyed collection with any key type (objects too), insertion-ordered, .size, real iteration. Prefer over plain objects for dynamic dictionaries.
  • Set — unique values; [...new Set(arr)] is the canonical dedupe.

Modules (import/export)

ES modules: export const x / export default and import { x } from './m.js'. Static, tree-shakeable, top-level await allowed. In Node, enable via "type": "module" or .mjs. (This demo uses CommonJS so it runs on any Node without config.)

Senior tip: Use ?? for value defaults (preserves 0/''), and ?. with ?? together: config?.timeout ?? 30 reads "the configured timeout, or 30".

🎯 Interview questions

Difference between || and ???

|| falls back on any falsy value (0, '', false); ?? only falls back on null/undefined. Use ?? for numeric and boolean defaults.

When does a destructuring default apply?

Only when the value is undefined. An explicit null is kept, not replaced by the default.

When choose Map over a plain object?

When keys aren't strings/symbols, when you need guaranteed insertion order, frequent add/delete, easy .size, or to avoid prototype-key collisions.

How do you dedupe an array?

[...new Set(arr)] — Set keeps only unique values, then spread back into an array.

What does obj?.a.b do if obj is null?

It short-circuits to undefined at obj?. without evaluating .a.b, avoiding a TypeError.
